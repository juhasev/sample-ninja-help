### Compensation grid by data variable

This feature allows you to define multiple compensation tables based on a radio data variable.

For example, in medical panels, compensation can now vary by **specialty** without requiring separate sub-panels for each audience segment.

**Key benefits**

- Configure compensation more precisely by respondent group
- Support specialized audiences with different incentive requirements
- Simplify compensation management

> **IMPORTANT:** You must ensure that each panelist has the data variable for which compensation is defined. You can do a quick check under **Data Variables** (click the blue stat button and add a subpanel filter).
