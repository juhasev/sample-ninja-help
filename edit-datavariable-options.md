### Personally Identifiable (PI)
You designate the question as Personally Identifiable, and it will not be displayed in all contexts. A user with **VIEW PI** permission can normally see the collected data for this data variable.

### Hidden from panelists
Makes the data variable hidden in the Member App or in the Mobile App (My account -> Collected Data) section.

### Community API writable
Turn on if this question needs to be Community API writable. This allows you to control which questions can be updated using the community API.

### Developer mode
Displays additional information for the developers, such as DataVariable UUID and Unique IDs.
